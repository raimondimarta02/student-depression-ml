# student-depression-ml
# Predicting Student Depression using Machine Learning

## Project Overview
This project analyzes a dataset to predict student depression using machine learning models.

## Structure
The full project (data + code + analysis) is contained in a single notebook file uploaded in this repository.

## Methodology
- Data exploration and preprocessing
- Feature engineering
- Model training:
  - Logistic Regression
  - KNN
  - Random Forest
  - Gradient Boosting
- Model evaluation using accuracy, precision, recall, and ROC-AUC

## Results
- Best model: Logistic Regression
- ROC-AUC: ~0.91
